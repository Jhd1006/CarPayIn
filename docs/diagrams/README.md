# Lucidchart Sequence Diagrams

These files are Mermaid sequence diagrams split for Lucidchart import.

Import them one at a time in Lucidchart's Mermaid diagram editor:

1. `lucid-01-qr-oauth-login.mmd`
2. `lucid-02-vehicle-confirmation.mmd`
3. `lucid-03-card-registration-billing-key.mmd`
4. `lucid-04-entry-parking-session.mmd`
5. `lucid-05-fee-payment-exit.mmd`

Each `.mmd` file contains one standalone `sequenceDiagram`.
